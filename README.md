# Tasteful Journeys Catering — Concept Dev Site

A concept marketing site for [Tasteful Journeys Catering](https://tastefuljourneyscatering.com/)
(Chef Alex Jones, Orlando/Casselberry FL), built around a single differentiating
brand position: **abundance / "we never run out of food."**

## The positioning

Instead of "delicious food for every occasion," the entire site leads with a named,
ownable promise competitors can't easily copy because it's operational, not just a slogan:

> **The Last Plate Promise™** — the first guest and the last guest deserve the same
> experience. When the guest count is confirmed, there will be enough food for every
> guest to enjoy a complete meal.

Positioning line: *Orlando's Most Generous Caterer.*

## Stack

Static site — `index.html`, `styles.css`, `script.js`. No build step. Deployed on Vercel.

Local preview:

```bash
python3 -m http.server 8747
# → http://localhost:8747
```
